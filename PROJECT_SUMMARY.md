# Meowtin.com - Personal Portfolio & Creative Showcase

## Project Summary: Meowtin.com Personal Portfolio

### **Project Overview**

This is a sophisticated **personal portfolio and creative showcase website** built as a **multi-faceted digital experience** that demonstrates expertise across **web development**, **3D graphics**, **interactive design**, and **creative content management**. The project serves as both a professional portfolio and a comprehensive showcase of creative work spanning **music production**, **video editing**, **digital art**, and **web development**.

### **Key Technologies Used**

#### **Frontend Technologies**

- **Next.js 15.2.4** - Modern React framework with App Router and static export
- **React 19** - Latest React with concurrent features and modern hooks
- **TypeScript 5** - Type-safe development throughout the application
- **Three.js 0.176.0** - 3D graphics and WebGL rendering engine
- **@react-three/fiber** - React renderer for Three.js
- **@react-three/drei** - Useful helpers and abstractions for Three.js
- **@react-three/postprocessing** - Post-processing effects (bloom, etc.)
- **Framer Motion 11.18.2** - Advanced animations and transitions
- **Tailwind CSS 3.4.17** - Utility-first CSS framework with custom design system
- **Lucide React** - Modern icon library
- **React Icons** - Comprehensive icon collection

#### **Styling & Design**

- **Custom CSS** - Advanced CSS with keyframe animations and 3D transforms
- **CSS Variables** - Dynamic theming system with light/dark mode support
- **Google Fonts** - Shadows Into Light handwritten font integration
- **Responsive Design** - Mobile-first approach with custom breakpoints
- **Custom Hooks** - useScreenCategory for responsive behavior

#### **Build & Development Tools**

- **Next.js Static Export** - Optimized for static hosting
- **Turbopack** - Fast development builds
- **PostCSS** - CSS processing and optimization
- **Autoprefixer** - Automatic vendor prefixing
- **ESLint** - Code linting and quality assurance

#### **Content Management**

- **JSON Configuration** - Structured data for projects and art gallery
- **File System API** - Server-side content reading for art gallery
- **Image Optimization** - Next.js Image component with WebP support
- **Video Integration** - Embedded YouTube videos and local video assets
